# Continuity Help Center

Step-by-step guides for every part of the tool. Each guide has CMS-ready text and screenshots in `en/img/` (the `[image: name.png]` placeholders mark where each one goes).

## Dashboard views
- [Today's View](today.md) — your home view: today's focus, alerts and progress.
- [Projects](projects.md) — list, filters, sort, project card and detail.
- [Tasks](tasks.md) — urgency buckets, filters, task row, completing and creating.
- [Routines](routines.md) — recurrences, completing occurrences and archiving.
- [Ideas](ideas.md) — ideas parking lot and promoting to a project.
- [Quick Notes](notes.md) — notebook with collapsible sections and a two-panel editor.
- [Log](log.md) — activity and updates timeline.
- [Analytics](analytics.md) — productivity panels and trends.
- [Graveyard](graveyard.md) — closed projects, autopsy and revive.

## Settings
- [Appearance (theme · colors · language)](appearance.md) — customize the look and language.
- [Plugins](plugins.md) — available integrations (overview, without the connection flow).
