Projects

The Projects tab is your master list: every project in one table, with its status, priority, category, task progress and next step. From here you search, filter, sort, open each project to see its detail, and create new ones. This guide walks through every part with a screenshot and explains how to use it and how to edit it.

How to use it

  • The search box filters by name, description, next step, motivation and category.
  • Tap a Status, Category or Priority chip to filter; go back to “All” to clear it.
  • Statuses with no projects aren't shown; the number on each chip is its count.
  • The Sort selector changes how the list is arranged (step 4).

How to edit it

  • Filters are cumulative: status + category + priority + due date at once.
  • If nothing matches, a “Clear filters” button appears to reset to “All”.

How to use it

  • Open it with the Filter button; its badge shows how many filters are active.
  • Pick one per group: Status, Priority, Category (if you have any) and Due date.
  • The Apply (N) button carries the live count; “Clear” resets back to All.

How to edit it

  • Due date: Any, Overdue, Next 7 days or No date.
  • On desktop these same filters live in the labeled bar (step 2), with no sheet.

What each mode does

  • Smart — overdue first, then due today, then idle, then by priority.
  • By priority — Critical → High → Medium → Low.
  • Recent activity — most recently touched on top.
  • Name — alphabetical.
  • By status — groups by status in its natural order.

How to edit it

  • The chosen mode persists while you stay in the view.

What it shows

  • Color stripe on the left = priority (red critical, orange high, green medium, blue low).
  • Square icon = project status (Active, Stalled, Idea, Launched…).
  • Category chip in its color, if the project has one.
  • Urgency badge: N overdue, N today, Nd idle or Nh pending.
  • → Next step under the name, and the done/total task bar on the right.

How to use it

  • Tap the row to expand the project detail (step 6).
  • The amber “idle” badge is just a visual hint (7+ days), not a status.
  • On mobile, category and idle hide to save space; progress collapses to “done/total”.

How to use it

  • Next step on top, always visible; below it Why this matters, Description, Tasks, Activity and Notes.
  • In Tasks: check the ✓ to complete, Add task, or edit/delete each one.
  • Log update leaves a log entry and “wakes” the project up.
  • Open project (↗) jumps to the full detail view.

How to edit it

  • Edit opens the project form (name, status, priority, category, dates…).
  • Delete removes the project (asks for confirmation).
  • Long task or activity lists are trimmed with a “show more”.

How to use it

  • On desktop, the header's New button opens the project form.
  • On mobile, the floating + button does the same (and offers to open Loop).
  • The empty state includes a direct shortcut: “Add your first project”.

How to edit it

  • If filters leave the list empty you'll see “No projects match these filters” with Clear filters.
  • A search with no results shows the searched text so you can adjust the term.
