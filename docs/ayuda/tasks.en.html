How to use it

  • Type in Search tasks or project… to filter by title or project name.
  • While searching, all groups expand automatically so you see every match.
  • If nothing matches, you'll see No tasks match “…”.
  • New opens the editor to create a task.

How to use it

  • Overdue — past due and not done. They come first, most-overdue at the top.
  • Today — due today. Always visible; its chip sums the estimated effort.
  • Upcoming — future due dates, sorted by date.
  • Pick a day — no date yet; here you give them one (see step 4).
  • Completed — the done ones; shows when you turn on “Show completed” in filters.

How to edit it

  • Tap a group's title to collapse or expand it. Your choice persists as you navigate.
  • The range (mobile) changes which groups show: Today keeps only Overdue + Today; Week adds Upcoming within 7 days; All shows everything.
  • A task moves between groups on its own based on its date and whether it's done.

How to use it

  • The border tints the state: red if overdue, orange if due today.
  • The ⏱ 2h chip is the estimated effort in hours (if you set one).
  • The bottom line shows the project and the date (or “Due today”).
  • 🔒 Blocked appears when the task has blockers; the row dims out.

How to edit it

  • Tap the task text to open its editor (title, project, date, effort, blockers).
  • In the Pick a day group, tap 📅 Add date to schedule it.
  • The ✕ on the right deletes the task (on desktop it appears on hover).

How to use it — Filters

  • By project — tap chips to include only those projects; No project catches loose tasks. No chips active = all.
  • Show completed — turns the Completed group on or off.
  • Show blocked — includes or hides tasks with blockers.
  • Clear resets the filter; Apply confirms it and closes the sheet.

How to use it

  • Tap the ✓ circle on the left to mark it done.
  • The row fades out and moves to the Completed group.
  • To undo, turn on “Show completed” in filters and tap the circle again (it's green) — it un-checks.
  • Completed tasks sort by completion time, most recent on top.

How to edit it

  • The Completed group stays reachable in any range, precisely so an accidental check can be undone.

Create & empty state

How to add your first task, and what you see when there are none yet.

Empty state + Create button
No tasks yet.
Create first task
▲With no tasks you'll see “No tasks yet.” and a button to create the first one. The floating + is always available.

How to use it

  • Create first task (empty state) or New (desktop) open the task editor.
  • On mobile, the floating + button is the main way in; it expands the “New task” action and access to Loop (the AI assistant).
  • On save, the task drops into the group it belongs to based on its date.

How to edit it

  • The floating + can't be hidden or reordered: it's a fixed part of the view.
  • To clear everything and return to the empty state, delete each task with its ✕.
