Today's View

The Today tab is the first thing you see when you open Continuity. It gathers, in one place, what deserves your attention today: tasks coming due, projects going cold, what you've already finished, and the projects that are one step away from closing. This guide walks through every section and explains how to use it and how to customize it.

  1. Header — today's date, a greeting based on the time, and the gear for Customize Today.
  2. Attention sections — alerts and focus: what's urgent stays on top.
  3. Progress sections — "Done today", "Close to the finish line", active/launched projects.
  4. Create button (FAB) — floating bottom-right, to add a task/project/routine/idea.
Sections only appear when they have data. If you have no stalled projects, the "Stalled alert" won't show; as you create your first project more blocks gradually appear. It's normal for a new user to see only a few sections at first.

How to use it

  • Click the ✓ circle on the left to mark the task done.
  • Click the task text to open its editor.
  • Click the project name to jump to it.
  • "View all tasks" opens the Tasks tab with the rest.

How to edit it

  • It's the only section that cannot be hidden (shown with 🔒 in the editor).
  • You can still move it in Customize Today.
  • Its content is filled from: dated tasks, project next steps, and stalled projects.
If nothing is urgent you'll see "Nothing pressing today." with a hint to add your first project or next steps. That means you're on top of things, not that something is broken.

Customize Today

Reorder, hide and show sections however you like. This is the "edit" for the whole view.

▲Each row = one section. Drag ⠿ to reorder; the eye 👁 shows/hides; the lock 🔒 marks the ones that can't be hidden.

Editor actions

  • Drag a row's ⠿ handle to change the order (keyboard works too).
  • Tap the eye 👁 to hide a section; tap again to show it. Hidden ones show a dashed border and a Hidden tag.
  • Rows with a lock 🔒 (e.g. "Today's Focus") can't be hidden, but can be moved.
  • The Mobile tag marks sections that currently show on phones only.

Save & reset

  • Done (green button) or the ✕ exit the editor and save your layout.
  • Reset ↩ returns the default order and visibility.
  • On exit, Today reloads to reflect what you showed/hid.
  • Your setup is saved to your account (it syncs with the mobile app).
