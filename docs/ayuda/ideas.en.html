Ideas Parking Lot

The Ideas tab is your parking lot: park new ideas here so you don't abandon what you're already doing, and promote them to a project when you're ready. It's the space between “having an idea” and “starting a project.” This guide walks the list, the idea card, how to capture, how to promote, and how to search — each with its own screenshot. This view's accent color is purple.

What it shows

  • The purple lightbulb 💡 and the idea's title (the only required part).
  • The reason in italics after an arrow (→): your “why it matters,” if you wrote one.
  • Free-form notes below, also optional.

How to edit it

  • Promote to project turns the idea into a brand-new project.
  • Edit opens the idea to change title, reason, or notes.
  • Delete drops it to make room for something new.
The reason and notes are hidden when empty: a card can be just a lightbulb and a title. Still, writing the “why” helps a lot when you return to the idea weeks later.

How to use it

  • Type a short title: that's enough to park the idea.

  • Capture saves it and adds it to the grid instantly.

How to edit it

  • The same modal is reused to edit (titled “Edit Idea”).
  • With no title, the Capture button stays disabled.
  • Cancel or the ✕ closes without saving.

Promote to project

When an idea is ready to start, turn it into a project with one tap.

Promote to project
Habit-tracking app
Habit-tracking appActive project
0/0 tasks · ready to plan
▲The idea leaves the parking lot and shows up as an active project in the Projects tab.

How to use it

  • Tap Promote to project on the card.
  • A project is created with the idea's title and reason as a starting point.
  • The idea leaves Ideas: it's no longer a “maybe,” it's a project.

How to edit it

  • From the project, add tasks, next steps, and hours.
  • Changed your mind? Manage the project from the Projects tab.
Promoting is one-way: the idea becomes a project. If you only meant to save it, use Edit instead of promoting.

How to use it

  • Type in the search box: it filters live, no search button.
  • It matches each idea's title, reason, and notes.
  • Clear the text to see all your ideas again.

How to edit it

  • There are no categories or tags: search is the only filter.
  • The app decides the order; you decide what to promote or drop.
